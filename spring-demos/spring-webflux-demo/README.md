# Spring Boot WebFlux Sample

### Notes
1. Seems @EnableWebMvc conflicts with @EnableWebFlux, because both WebMvcConfigurationSupport and WebFluxConfigurationSupport class will instantiate a new bean in RequestMappingHandlerAdapter type. So, it will throw an IllegalStateException.
And I cannot google an answer to solve this problem, so I just simply drop spring-web-mvc dependency.
